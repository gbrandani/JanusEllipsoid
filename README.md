JanusEllipsoid
==========

## Description

A Monte Carlo simulation and visualization tool for studying the adsorption of a Janus ellipsoidal colloid at a fluid interface.

## Build and run

```sh
$ cd src
$ javac *.java
$ java InteractiveJanusEllipsoid
```
